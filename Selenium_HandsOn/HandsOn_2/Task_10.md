# Task 10 — Test Artifacts per SDLC/TDLC Phase Pair

| SDLC Phase | TDLC Phase | Test Artifact Produced |
|---|---|---|
| Requirements | Acceptance Testing | Acceptance Test Plan is prepared during this phase |
| System Design | System Testing | System Test Plan, covering end-to-end scenarios across modules |
| Architecture Design | Integration Testing | Integration Test Plan, covering interfaces between components (e.g. API <-> DB) |
| Module Design | Unit Testing | Unit Test Cases / Test Design Specifications for individual functions and classes |
