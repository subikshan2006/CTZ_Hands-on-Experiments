# HandsOn 4 — Selenium Setup & Basic Commands

First Selenium scripts against the LambdaTest Playground: opening the browser,
navigation, window/tab handling, and screenshots.

## Setup
```bash
pip install -r requirements.txt
```
Needs Chrome installed locally.

## Run
```bash
python Task_25.py   # open, print title, close
python Task_26.py   # implicit wait
python Task_27.py   # headless mode
python Task_28.py   # navigate + back()
python Task_29.py   # new tab handling
python Task_30.py   # screenshot
python Task_31.py   # window sizing
```
`Task_24.py` is just notes on Selenium's architecture, no browser involved.

`Task_30.py` writes `playground_screenshot.png` into this folder.
